Pour réalisé ce projet nous avions utilisé le logiciel Cobian Backup afin de mieux comprendre les différentes sauvegardes.


Qu'est-ce qu'est Cobian Backup ?

Cobian Backup est un logiciel permettant de réaliser des sauvegardes complètes, incrémentales ou différentielles et de programmer le démarrage automatique des tâches de sauvegarde.
En outre, Cobian Backup supporte la compression de données et le chiffrement.


La sauvegarde complète :

         Il s'agit de réaliser une copie totale de toutes vos données qui sont transmises.
Schèma illustrant la sauvegarde complète

La sauvegarde incrémentale :

         La sauvegarde incrémentale permet uniquement de sauvegarder les fichiers modifiés depuis dernière la sauvegarde précédente.
Schèma illustrant la sauvegarde incrémentale

La sauvegarde différentielle :

         La sauvegarde différentielle copie toutes les données depuis le dernier backup incrémental ou complet.
